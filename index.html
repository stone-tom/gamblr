Gamblr

Gamblr is a simple Web app, to store "Hosn Owi" game stats. The App works offline. Old games get cached, so if you are offline, you can see them too. When creating new games, they get saved in Local Storage, and can be sent to the Backend, when you are online again.

Gamblr PWA Buttons

The Push notifications can only be testet locally, because the backend cant send Push Notifications yet.

If you want to test, you have to clone the repo and change the "applicationServerPublicKey" variable in the script.js file and send notifications with the following link "Push Companion"
